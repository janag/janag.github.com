Home

Sencha Touch
Home

Sencha Touch is one of the very first enterprise-ready HTML5 Mobile Web frameworks. The Mobility team will present how the MVC pattern was used in developing the GetThere Mobile Shop and Book application.

Perfecto Mobile
Home

Much of the mobile applications are still tested manually using the available emulators and phones. Automation testing on the actual phones on the real carrier network is taking shape. Perfecto Mobile is one of the pioneers. This session will show how to automate mobile application testing to increase the development velocity and quality.

Titanium
Home

A free and open source application development platform, Titanium lets you create native mobile, tablet and desktop application experiences using existing web skills like Javascript, HTML, CSS, Python, Ruby, and PHP

PhoneGap
Home

PhoneGap is a cross-platform development framework that enables developers to use HTML5 to deliver applications to iOS, Android, BlackBerry, webOS and Symbian platforms. PhoneGap applications are native applications, under the hood leveraging a platform's browser to deliver content and compelling user experiences. At the same time, developers can access any functionality native to each platform with the help of a JavaScript API. Write once, run (almost) anywhere.

Perfecto Mobile
Home

PerfectoMobile is formerly known as Nexperience, is a leading provider of remote access and automated testing solutions for mobile devices. The company was founded in 2006 by a group of highly experienced executives from the mobile and telecom industries. Perfecto Mobile enables consumer serving enterprises, mobile operators, device manufacturers, and mobile developers, to significantly increase the quality of their products and on-going services while reducing time-to-market, costs and logistical efforts. By providing automated and manual remote access to real mobile devices around the globe, the company's MobileCloud platform dramatically improves efficiency of operations in the various stages of the mobile product life cycle, including development, testing, porting, deployment, and on-going monitoring in support of commercial services.
